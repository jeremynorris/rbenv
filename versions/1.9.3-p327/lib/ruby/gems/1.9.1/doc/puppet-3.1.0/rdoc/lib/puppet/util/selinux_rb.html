Description

Provides utility functions to help interface Puppet to SELinux.

This requires the very new SELinux Ruby bindings. These bindings closely mirror the SELinux C library interface.

Support for the command line tools is not provided because the performance was abysmal. At this time (2008-11-02) the only distribution providing these Ruby SELinux bindings which I am aware of is Fedora (in libselinux-ruby).
